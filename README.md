# AirBnB Clone - Command Interpreter

## 📌 Project Description

This project is the first step toward building a full web application: an AirBnB clone.  
It focuses on creating a command-line interpreter to manage objects such as users, places, and more.

The command interpreter allows you to:
- Create new objects
- Retrieve existing objects
- Update object attributes
- Delete objects
- Persist data using JSON serialization

This project lays the foundation for:
- Object-oriented programming
- File storage systems
- Serialization/deserialization
- Unit testing
- Future integration with databases and web frameworks

---

## 🧠 Command Interpreter

The command interpreter is built using Python's `cmd` module.  
It behaves similarly to a Unix shell but is tailored to manage project objects.
