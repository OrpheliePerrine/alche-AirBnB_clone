# AirBnB Clone - Command Interpreter

## 📌 Project Description

This project is the first step towards building a full web application: an AirBnB clone. It focuses on creating a command-line interpreter to manage objects for the application.

The goal is to understand how to build a scalable system by implementing object-oriented programming, serialization/deserialization, file storage, and unit testing.

This command interpreter will be used in future stages of the project, including HTML/CSS templating, database storage, API development, and front-end integration.

---

## 🧠 Learning Objectives

By completing this project, you should be able to:

* Create a Python package
* Build a command interpreter using the `cmd` module
* Understand and implement unit testing
* Serialize and deserialize Python objects
* Read and write JSON files
* Work with `datetime`
* Generate unique IDs using `uuid`
* Use `*args` and `**kwargs`
* Handle named arguments in functions

---

## ⚙️ Command Interpreter

### What is it?

The command interpreter is similar to a Unix shell, but it is designed specifically to manage objects in this project.

It allows you to:

* Create new objects (e.g., User, Place)
* Retrieve objects
* Update object attributes
* Delete objects
* Display all objects

---

## ▶️ How to Start

1. Clone the repository:

```bash
git clone https://github.com/your-username/AirBnB_clone.git
cd AirBnB_clone
```

2. Make the console executable:

```bash
chmod +x console.py
```

3. Run the command interpreter:

```bash
./console.py
```

You should see the prompt:

```
(hbnb)
```
